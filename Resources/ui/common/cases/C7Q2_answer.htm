Recognizing that PenguinPro is at risk of being overtaken by other games, the CEO decides to spend money on advertising to acquire new users. He has 2 options, which is best?

  • Facebook: Approx 100k clicks/day, 10% conversion rate
  • Magazine: Approx 1M reader base
  • Game cost $1M to develop

Answer 2

Basic Points

  • Cost/New User on Facebook $0.6. Need the user to play the game for at least 10 days to breakeven on the advertising investment
  • Potential to acquire 10k new users a day via Facebook. This would nearly double the rate of user acquisition
  • If adverts are performing poorly on Facebook you stop the campaign. Likely cost $6k per day
  • With game magazine it's a one off cost and don't know how many readers will get the game

Advanced Comments

  • Sunk Costs: Game development costs are a sunk cost. Should now be thinking about marginal profit and should be prepared to spend up to the full known value of a user to acquire
  • Growth: This is a fast growing environment like a land grab so it's important to grow user base quickly. Rely on the assumption that revenue and profit can be generated in time once loyal users are acquired.
