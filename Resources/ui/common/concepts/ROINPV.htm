Return on Investment (ROI)

= The amount that is earned from a given investment activity, divided by the amount invested

The difference between the present value of cash inflows and the present value of cash outflows.


Net Present Value (NPV)

NPV is used in capital budgeting to analyze the profitability of an investment or project. If a project is forecast to be NPV positive, then this indicates that it will be profitable to proceed.

  • NPV = X / (1+r)^n
  • r = discount rate
  • n = number of years
